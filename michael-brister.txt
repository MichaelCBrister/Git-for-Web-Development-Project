    8. What elements can I add an id to?
         any html element
   
    9. How many times can I use the same id on a page?

            a class can be used to style multiple elements while an id can only represent one element. Use a class if you have multiple elemts that you would like to have the same style and an id for one specific element

            selects every one of a specified element with the same parent element. this would be useful to style multiple elements found within the same parent element
